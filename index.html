Demo of SimpGraph (simple graph)

Once upon a time, I want something. But, to do that having a graph is a good accesories. I will use graph more but i don't want my time taken to build table, as i am coding html using gedit(notepad). So I created this SimpGraph, just a simple engine to read div and its data attribute then build graph.

<div class="simpgraph"
data-sg-type="bar-horizontal-css"
data-sg-width="400"
data-sg-height="100"
data-sg-value="0.10,0.24,0.52,0.78,0.44,0.98">
</div>

</div>
